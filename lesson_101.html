Activity 2: Drawing a Webpage (15 minutes)

  1. Let RAIE draw their own webpage design - encourage creativity!

Activity 3: What is HTML? (10 minutes)

  1. Introduce HTML: "RAIE, HTML is like the skeleton of a webpage! Just like your body has bones to give it shape, webpages have HTML to give them structure."
  2. Explain the acronym:
    • HyperText - Special text that can link to other pages
    • Markup - Instructions that tell the computer how to display things
    • Language - A way for us to "talk" to computers
  3. Use an analogy: "Think of HTML like instructions for building with LEGO blocks. We tell the computer: 'Put a big title here, add a picture there, make this text bold!'"

Activity 4: HTML Tags - The Magic Instructions (15 minutes)

  1. Introduce tags: "HTML uses special codes called 'tags' - they're like magic words that tell the computer what to do!"
  2. Show the basic structure:
<!DOCTYPE html> <html> <head> <title>RAIE's First Website</title> </head> <body> <h1>Hello, I'm RAIE!</h1> <p>Welcome to my first webpage!</p> </body> </html>
  1. Explain each part:
    • <!DOCTYPE html> - "Hey computer, this is an HTML page!"
    • <html> - The main container (like a big box)
    • <head> - Information about the page (like a label on the box)
    • <title> - What shows in the browser tab
    • <body> - Everything people see on the page
  2. Point out: "See how tags come in pairs? Like bookends holding books together!"

Activity 5: Hands-On Coding Time! (15 minutes)

  1. Open Replit.com and create a new HTML project
  2. Say: "Now RAIE, let's build your very first webpage together!"

üí° Teaching Tips for Success

üéØ Keep RAIE Engaged:
